Автоматическое определение местоположения С помощью датчиков, установленных в телефоне, и информации в базе данных определяется текущее местоположения обходчика. Автоматически определяются: жилой комплекс, дом, секция, этаж и квартира
Кросс-платформенность Произведите видеосъёмку в нашем приложении и посмотрите полученную информацию на любом удобном устройстве: на телефоне через мобильное приложение; на любом другом устройстве через сайт.
Интерфейс для каждого Интуитивно понятный интерфейс позволяет не тратить много ресурсов на внедрение и обучение сотрудников. Начать работать с программой можно самостоятельно.
Возможности системы для строительства
Вся информация об отделке на одном сайте
    Сайт позволяет быстро и удобно получать информацию о ходе отделочных работ в каждом жилом комплексе. Для каждой квартиры Вы можете посмотреть:
  • информацию о ходе отделочных работ
  • видео обхода данной квартиры.
Контроль отделки в Вашем телефоне Снимите процесс обхода этажа с помощью нашего приложения и получите мгновенный результат анализа готовности отделки поквартирно. Информация о результатх обхода будет храниться в базе данных и на сервере, поэтому Вы также сможете посмотреть ее через наш сайт.
Аналитика под рукой На сайте представлен раздел “Анализ” для каждого жилого комплекса, где Вы можете скачать “шахматки”, диаграммы и графики для дальнейшего анализа хода отделочных работ.
